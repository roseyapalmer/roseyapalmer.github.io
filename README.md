## Running locally

### Requirements

- Ruby > 2.4
- Bundler > 2.0
- Jekyll > 4.0

Once you've cloned the repository:

```
# Install project dependencies
bundle install

# Run a local server to preview your work 
bundle exec jekyll serve
```
